## Run Locally

Clone the project

```bash
  git clone https://github.com/phyohan18/Automation-Tool-for-Chrome-Extension.git
```

Go to the chrome extension setting

```bash
  - Make sure dev mode is on
  - Click on load unpacked 
  - Select the project folder

```
